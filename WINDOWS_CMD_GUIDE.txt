# Windows cmd.exe Quick Reference for Space Habitats RAG

## Setup (First Time)

1. Extract backend-refactored.zip to C:\Projects\

2. Open Command Prompt (cmd.exe)

3. Navigate:
   cd \Sandbox\rag-space-habitats\backend-refactored

4. Install:
   npm install

5. Configure:
   copy .env.example .env
   notepad .env
   (Edit with your settings, save, close)

6. Database:
   mysql -u root -p space_habitats_rag < migrations\001_add_query_log.sql

7. Start:
   npm run dev

## Daily Use

Start server:
   cd \Sandbox\rag-space-habitats\backend-refactored
   npm run dev

Stop server:
   Press Ctrl+C

Restart server:
   Press Ctrl+C, then: npm run dev

## Access Points

Frontend:    http://localhost:3000
API:         http://localhost:5000
AdminJS:     http://localhost:5000/admin

## Troubleshooting

Port already in use:
   netstat -ano | findstr :5000
   taskkill /PID <number> /F

Check MySQL running:
   mysql -u root -p

View logs:
   (They appear in the cmd.exe window)
